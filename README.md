### [+] Description
This tool is designed to simulate payment flows on
Stripe & WooCommerce-based checkout systems.
It follows real browser logic, session handling,
and response-based classification (LIVE / DEAD / RETRY).

### [+] Installation Termux, Linux, Ubuntu
```
- termux-setup-storage
- apt update && apt upgrade -y
- apt install git python python2 nano
- git clone https://github.com/KianSantang777/MeduzaV3.git
- cd MeduzaV3
- pip install -r requirements.txt
- python meduzav3.py
```
